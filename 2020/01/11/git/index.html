# GIT的常用命令

git 的命令有很多，下面列出了较为常用的 基本命令大全。

# Git 克隆本地：

---
git clone git@code.aliyun.com:damonwu/wxapp-magazine.git  
---

# 切换本地开发分支 （拉取分支代码）：

---
git checkout develop 
---

# 添加修改的文件到暂存区：

---
git add   
---

# 克隆本地提交描述：

---
git commit -m "xxxx"
---

# 提交到远程仓库（develop自动对应develop：

---
git push
---

# 切换本地master分支：

---
checkout master
---

# 合并本地master分支与开发分支,-no-ff保留分支历史：

---
git merge develop --no-ff 
---

# 切换本地开发分支：

---
git checkout develop
---

# 下载指定分支 （如分支已有，则用这行代码拉取最新代码，git pull origin develop）：

---
git pull <remote> <branch>
---

# 查看本地配置信息：

---
git config --list
---

# 查看分支日志：

---
git log
---
