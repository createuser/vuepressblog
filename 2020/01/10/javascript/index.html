# JavaScript高级程序设计（第3版）学习笔记

下面是学习JavaScript高级程序设计的一些笔记

# JavaScript组成部分

① ECMAScript: 提供核心语言功能；
② 文档对象模型（DOM）:提供访问和操作网页内容的方法和接口；
③ 浏览器对象模型（BOM）:提供与浏览器交互的方法和接口；

# Script脚本

① 引用外部Javascript文件，使用src属性指向文件URL。可同一服务器或其他域文件。
② 所有`<script>`按照页面出现顺序加载
③ `defer`: 让脚本在文档完全呈现后执行,只适用于外部文件
④ `async`: 立即下载文件，不会阻塞文档呈现。只适用于外部文件。不能保证异步脚本按照它们在页面中出现的顺序执行

# 数据类型

5种简单数据类型：Undefined、Null、Boolean、Number、String; 1种复杂数据类型：Object

① typeof 操作符: 
对一个值使用typeof操作符可能会返回下列某个字符串：
undefined - 未定义
boolean - 布尔值
string - 字符串
number - 数值
object - 对象 或 null
function - 函数
② undefined ： 在使用var或let 声明变量但未对其加以初始化时，这个变量的值就是undefined。
注意：对未初始化和未声明的变量执行typeof操作符均返回undefined。无论在什么情况下都没有必要把一个变量的值显式地设置为undefined。
③ Null 类型：空对象指针，undefined 值派生自null值。
④ Boolean 类型
对任何数据类型的值调用Boolean()函数，返回一个Boolean值
Number：任何非零数字值（包括无穷大）=> true, 0和NaN =>false
